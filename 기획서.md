# 사진 모자이크 & 강조 편집 웹툴 기획서

## 1. 개요

클립보드에 복사된 이미지를 붙여넣기(Ctrl+V)하면 즉시 편집 화면이 열리고, 사람 얼굴 등 민감한 영역을 **모자이크 처리**하거나 사진 속 특정 부분을 **도형으로 강조**한 뒤, 결과물을 다시 **클립보드로 복사**하거나 **이미지 파일로 저장**할 수 있는 단일 페이지 웹 애플리케이션.

- **목표 사용자**: 캡처 이미지에서 개인정보(얼굴·번호판·이름 등)를 가리거나, 문서/매뉴얼용으로 특정 영역을 강조해야 하는 일반 사용자
- **핵심 가치**: 설치 없이 브라우저에서 즉시, 붙여넣기 → 편집 → 복사/저장까지 빠른 워크플로우
- **형태**: 클라이언트 사이드 단일 웹 페이지 (서버 업로드 없음 → 프라이버시 보장)

## 2. 핵심 기능

### 2.1 이미지 입력
| 방식 | 설명 |
|------|------|
| 클립보드 붙여넣기 | `Ctrl+V` / `Cmd+V` 로 클립보드 이미지를 캔버스에 로드 (주 입력 방식) |
| 드래그 앤 드롭 | 파일을 캔버스 영역에 끌어다 놓기 |
| 파일 선택 | 버튼 클릭 → 파일 탐색기로 이미지 선택 |

입력 시 캔버스 크기를 이미지 해상도에 맞게 자동 조정하고, 화면에는 축소 표시하되 원본 해상도로 편집·저장.

### 2.2 모자이크(가리기) 기능
사진 속 얼굴 등 민감 영역을 흐리게/픽셀화 처리.

- **영역 지정 방식**
  - 사각형(네모) 드래그
  - 원형(동그라미) 드래그
  - 자유 곡선(올가미/브러시) — 마우스로 자유롭게 영역 그리기
- **처리 방식 선택**
  - 픽셀화(모자이크) — 블록 크기 조절 슬라이더
  - 블러(가우시안) — 강도 조절 슬라이더
  - 단색 채우기(검정 박스 등) — 옵션
- (선택) 자동 얼굴 감지: 라이브러리로 얼굴을 찾아 원클릭 모자이크 — *2차 기능*

### 2.3 강조(도형) 기능
사진 속 특정 장면을 속이 빈 도형으로 강조.

- **도형 종류**
  - 빈 사각형
  - 빈 원/타원
  - 화살표
  - (선택) 자유선, 직선
- **스타일 옵션**
  - 선 색상(컬러 피커)
  - 선 두께(슬라이더)
  - (선택) 외부 어둡게 처리(스포트라이트 효과)

### 2.4 공통 편집 기능
- 객체 선택 / 이동 / 크기 조절 / 삭제
- 실행 취소(Undo) / 다시 실행(Redo)
- 전체 초기화
- 레이어 순서(앞으로/뒤로) — *선택*

### 2.5 출력
| 기능 | 설명 |
|------|------|
| 클립보드로 복사 | 편집 결과를 다시 클립보드에 이미지로 복사 (`Clipboard API`) |
| 이미지 저장 | PNG / JPG 다운로드 |
| (선택) 품질·포맷 선택 | 저장 시 포맷과 품질 지정 |

## 3. 화면 구성 (UI)

```
┌──────────────────────────────────────────────┐
│  [상단 툴바]                                    │
│  선택 | 사각형모자이크 | 원모자이크 | 자유모자이크 │
│  | 사각형강조 | 원강조 | 화살표                  │
│  ─────────────────────────────────────────     │
│  처리방식: [픽셀화 ▾] 강도:[━━●━━]              │
│  색상:[■] 두께:[━●━] | Undo Redo 초기화          │
│  ─────────────────────────────────────────     │
│  [복사] [저장 ▾]                                │
├──────────────────────────────────────────────┤
│                                                │
│            [ 캔버스 / 편집 영역 ]                │
│       (비어있을 때: "Ctrl+V로 붙여넣기")         │
│                                                │
└──────────────────────────────────────────────┘
```

- 이미지가 없을 때: 안내 플레이스홀더 표시 (붙여넣기/드롭/파일선택 유도)
- 도구 선택 시 커서 변경 및 활성 도구 하이라이트

---

## 4. 사용자 흐름 (User Flow)

1. 사용자가 다른 곳에서 이미지를 캡처/복사
2. 웹페이지에서 `Ctrl+V` → 캔버스에 이미지 로드
3. 도구 선택 (모자이크 or 강조)
4. 캔버스에서 드래그하여 영역/도형 적용
5. 필요 시 객체 이동·수정·삭제, Undo/Redo
6. 완료 후 [복사] 또는 [저장] 클릭
7. 결과물을 원하는 곳에 붙여넣기 / 파일로 보관

## 5. 기술 스택 제안

| 영역 | 선택지 |
|------|--------|
| 프레임워크 | React (또는 바닐라 JS) |
| 캔버스/그래픽 | HTML5 Canvas, 또는 Fabric.js / Konva.js (객체 기반 편집에 유리) |
| 입력 | Clipboard API (`paste` 이벤트), Drag & Drop API, File API |
| 출력 | `canvas.toBlob()`, Clipboard API (`navigator.clipboard.write`) |
| 모자이크 처리 | Canvas 픽셀 조작 / StackBlur 등 블러 라이브러리 |
| (선택) 얼굴 감지 | face-api.js, MediaPipe 등 |

> 객체를 선택·이동·수정해야 하므로 **Fabric.js** 또는 **Konva.js** 사용을 권장. 모자이크는 해당 영역을 잘라 픽셀화/블러 후 이미지 객체로 덮는 방식.

## 6. 주요 기술 고려사항

- **클립보드 권한**: 복사 기능은 HTTPS + 사용자 제스처 필요. 브라우저별 지원 차이 안내.
- **해상도 유지**: 화면 표시 크기와 별개로 원본 해상도 기준으로 편집·출력.
- **모자이크 복원 방지**: 단순 블러는 복원 가능성이 있으므로, 민감 정보는 강한 픽셀화 또는 단색 채우기를 기본 권장.
- **프라이버시**: 모든 처리는 브라우저 내에서만 수행, 서버 전송 없음 — 명시적으로 안내.
- **반응형**: 다양한 화면 크기 대응, 캔버스 자동 스케일.

## 7. 개발 우선순위 (MVP → 확장)

### MVP (1차)
- [ ] 클립보드 붙여넣기로 이미지 로드
- [ ] 사각형·원형 모자이크(픽셀화)
- [ ] 빈 사각형·원·화살표 강조 도형
- [ ] 클립보드 복사 / PNG 저장

### 2차 확장
- [ ] 자유 곡선(올가미) 모자이크
- [ ] 블러/단색 등 처리 방식 선택, 강도 조절
- [ ] Undo/Redo, 객체 선택·이동·삭제
- [ ] 드래그앤드롭·파일선택 입력

### 3차 (고급)
- [ ] 자동 얼굴 감지 후 원클릭 모자이크
- [ ] 스포트라이트(주변 어둡게) 강조 효과
- [ ] 저장 포맷/품질 옵션, 레이어 관리
