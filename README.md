# 🖼️ 사진 모자이크 & 강조 편집기

클립보드 이미지를 붙여넣어 **모자이크로 가리고**, 중요한 부분을 **도형으로 강조**한 뒤 다시 **복사 / 저장**할 수 있는 단일 페이지 웹앱입니다. 설치 없이 브라우저에서 즉시 사용할 수 있습니다.

> 🔒 모든 처리는 브라우저 안에서만 수행되며 서버로 이미지가 전송되지 않습니다. (얼굴 자동 감지 기능만 최초 1회 모델 로딩에 인터넷이 필요)

## 기능
- **입력**: 클립보드 붙여넣기(Ctrl/Cmd+V), 드래그앤드롭, 파일 선택
- **모자이크**: 사각형 · 원형 · 자유곡선 / 픽셀화 · 블러 · 단색채우기 (강도 조절)
- **강조**: 빈 사각형 · 빈 원 · 화살표 (색상 · 두께)
- **고급**: AI 얼굴 자동 모자이크, 스포트라이트(주변 어둡게)
- **편집**: 선택 · 이동 · 크기조절 · 삭제, Undo/Redo, 레이어 순서
- **출력**: 클립보드 복사, PNG / JPG 저장(품질 옵션)

## 단축키
`V` 선택 · `R` 사각모자이크 · `O` 원모자이크 · `F` 자유모자이크 · `A` 화살표 · `Ctrl+Z/Y` Undo/Redo · `Del` 삭제

## 배포
정적 사이트입니다. 빌드가 필요 없으며 `index.html` 하나로 동작합니다. Vercel / Netlify / GitHub Pages 등 어디서나 호스팅 가능합니다.
